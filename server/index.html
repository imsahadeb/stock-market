API Documentation

MongoDB Router

Endpoint Method Full URL Description
/stocks GET http://api.ghonta.in/api/stocks Retrieves all stocks
/stocks/search?q=your_query GET http://api.ghonta.in/api/stocks/search?q=your_query Searches for stocks based on a query
